# Qigong Infused Yoga

A Next.js website for Qigong Infused Yoga, featuring a calming light blue theme and comprehensive information about weekly Qigong classes.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build for Production

```bash
npm run build
npm start
```
